# 🎓 Student Management System

A full-stack Student Management System built using React, Django REST Framework, and MySQL.

## Features

- Add Student
- Edit Student
- Delete Student
- Search Students
- Pagination
- Dark Mode
- Course Analytics Dashboard
- Bar Chart
- Pie Chart
- Doughnut Chart
- Export CSV
- Export PDF
- JWT Authentication
- Responsive UI

## Tech Stack

### Frontend
- React
- Vite
- Axios
- React Icons
- React Toastify
- Chart.js
- React ChartJS 2
- jsPDF
- jspdf-autotable

### Backend
- Django
- Django REST Framework
- Simple JWT
- MySQL
